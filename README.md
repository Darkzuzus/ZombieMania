# Bienvenue sur le projet ZombieMania

ZombieMania est un jeu indépendant actuellement en cours de développement.

Il s'agit d'un FPS de survie contre des zombies. Pour le moment, vous devez survivre à des vagues infinies de zombies sur une carte relativement simple.

Pour télécharger le jeu, il vous suffit de télécharger l'installeur :

https://github.com/Darkzuzus/ZombieMania/releases/download/Latest/Installer.exe

Vous retrouverez également sur GitHub le code source de l'installeur ainsi que celui du Starter, qui permet de maintenir automatiquement le jeu à jour.

Si vous rencontrez des problèmes en jeu, avec l'installeur, le système de téléchargement automatique, ou si vous souhaitez simplement faire des suggestions, je vous invite à ouvrir une Issue sur GitHub ou à rejoindre le serveur Discord :

## Information importante concernant Windows Defender

L'exécutable de l'installeur ne possède actuellement pas de certificat de signature numérique. Il est donc possible que Windows Defender affiche un avertissement indiquant que le programme est inconnu.

Pour lancer l'installeur malgré cet avertissement :

1. Cliquez sur **« Informations complémentaires »**.
2. Cliquez ensuite sur **« Exécuter quand même »**.

Cette alerte est normale et est uniquement due à l'absence de signature numérique du programme.
